# 农户端使用指南
## 1. 账号登录与实名认证流程
### 登录操作
- 打开数字农人农户端小程序/APP，选择「微信快捷登录」；
- 登录后系统自动校验微信实名信息，无需额外输入账号密码。

### 实名认证
- 入口：首页「我的」→「实名认证」；
- 所需材料：仅需身份证正反面照片/身份证号（系统自动核验）；
- 注意：实名信息需与微信支付实名一致，无需绑定本人手机号。

## 2. 数农码激活与收款操作
### 数农码激活
- 入口：首页「数农码」→「激活码」；
- 操作：输入平台发放的激活码，点击「确认激活」即可；
- 未激活数农码也可收款：首页直接展示「待收款订单」，点击「快捷收款」完成操作。

## 3. 农产品合格证生成/签发教程
### 生成流程
- 入口：首页「合格证」→「新增合格证」；
- 填写信息：选择农产品品类、产地、批次、数量；
- 签发：确认信息无误后点击「生成合格证」，系统自动生成电子合格证。

## 4. 常见操作问题排查
- Q：登录提示“实名信息不一致”？
  A：核对微信实名与备案信息，确保姓名、身份证号完全匹配；
- Q：合格证生成失败？
  A：检查农产品品类选择是否正确，批次/数量是否填写完整；
- Q：收款后未到账？
  A：资金会在T+1个工作日到账，可在「我的」→「资金明细」查看进度。
